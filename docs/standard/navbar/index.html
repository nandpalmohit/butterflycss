Navbar

A Navbar is basically branding, list of links, buttons, dropdowns and menu with responsive navigation.

Use the <nav class="navbar"> tag to create the footer element.

Example

Navbar is a part of navigation bar that holds the links, forms, buttons, icons and more documentation with responsive layout.

The below Navbar is an example of butterflycss designed navbar which holds the links, dropdowns, buttons, forms with multiple color options.

<nav> tag is neccessary to create a Navbar component.

								<nav class="nav-xl nav-navy text-default cn-between">
								  <!-- navbar brand -->
								  <div class="brand">
								    <a class="navbar-brand Grand bold-400 size-22" href="#">Butterflycss</a>
								  </div>
								  <!-- label with checkbox -->
								  <label class="toggle" for="NavbarToggle">
								    <i class="fas fa-grip-lines"></i>
								  </label>
								  <input type="checkbox" class="navbar-toggle" id="NavbarToggle">
								  <!-- navbar links -->
								  <div class="navbar Sans">
								    <a class="navbar-link" href="#">Home</a>
								    <a class="navbar-link" href="#">About us</a>
								    <!-- navbar dropdown -->
								    <div class="dropdown">
								      <a class="navbar-link" href="#">Components</a>
								      <ul class="dropdown-content bg-navy text-light">
								        <li><a class="dropdown-link" href="#">Buttons</a></li>
								        <li><a class="dropdown-link" href="#">Cards</a></li>
								        <li><a class="dropdown-link" href="#">Footer</a></li>
								        <li><a class="dropdown-link" href="#">Navbar</a></li>
								      </ul>
								    </div>
								    <a class="navbar-link" href="#">Resources</a>
								  </div>
								  <button class="btn cl-btn-light Sans-Bold mx2" href="#">Download</button>
								</nav>

Navbar can holds multiple links, icons, buttons, form with wide range of colors, sizes and layouts. Butterflycss provide pre designed responsive layout with 4 different ranges.

You need to add <nav> to create a navbar. Navbar is created with flex so you can change the align with predefined classes like .cn-* for it.

Brand Logo

Brand is first choice in Navbar and it can be applied with a tag which works better fot it.

You need to add .brand div class to create a brand styling & add .navbar-brand class inside it to apply brand style and it is working with anchor tag and heading tag only.

Butterflycss 

								<nav class="nav-light text-black">
								  <div class="brand">
								    <img src="..." width="" alt="">
								    <a class="navbar-brand size-24">Butterflycss</a>
								  </div>	
								</nav>

The .brand class allows you to insert images, heading, anchor tag with inline position.

You can change the background color as well as text color with multiple options.

Links

Links are more preferred for footers, navigations, social media links and more.

You need to add .link-data class to apply link effect. It is only working with anchor tag.
