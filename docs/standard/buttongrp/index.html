Button Group

Used to perform multiple events, paging, groups with more styles, colors and sizing.

Use the .btn-grp class to create the button group.

Colored Button Group

Button groups are more preferred with colored buttons to attract the group.

First add .btn-grp class to create the button group and then simply add colored button class (.btn .cl-btn-*) to create the colored button group.

								<div class="btn-grp">
								<button class="btn cl-btn-default"> First </button>	
								<button class="btn cl-btn-default"> Second </button>

								<!--- secondary --->

								<button class="btn cl-btn-secondary"> First </button>	
								<button class="btn cl-btn-secondary"> Second </button>

You can use it with <a>, <button> tag easily. Also note that button group is mainly made for multiple events, paging numbers, groups so you can customize the group styling.

Link Button Group

Link Group is more usefull for menubar, sidebar, navbar, footer links.

First add .btn-grp class to create the button group and then simply add colored button class (.btn .btn-link) to create the colored button group.


Link 1 Link 2 Link 3 Link 4 

								<!---  button link --->
								<div class="btn-grp">
								<button class="btn btn-link"> Button 1</button>	
								<button class="btn btn-link"> Button 2</button>
								<button class="btn btn-link"> Button 3</button>
								<button class="btn btn-link"> Button 4</button>
								</div>
								<!--- a tag link --->
								<div class="btn-grp">
								<a class="btn btn-link"> Link 1</a>	
								<a class="btn btn-link"> Link 2</a>
								<a class="btn btn-link"> Link 3</a>
								<a class="btn btn-link"> Link 4</a>
								</div>

Link Buttons use as transparent button links and you can use it with <a>, <button> tag. <button> performs hoverable, active state while <a> tag set transparent background as default for all the state.
You can change text color as well as background color for links.

Ripple Button Group

Ripple Button Group use for individual button effects.

You need to add .ripple class to apply Ripple effect.
Add ripple-for="dark" attribute for dark ripple effect.
Add ripple-for="light" attribute for light ripple effect. 


								<!--- Ripple for colored --->
								<div class="btn-grp">
								<button class="btn ripple cl-btn-primary" ripple-for="light"> First </button>	
								<button class="btn ripple cl-btn-primary" ripple-for="light"> Second </button>
								</div>
								<!--- Ripple for outline --->
								<div class="btn-grp">
								<button class="btn ripple ot-btn-primary" ripple-for="dark"> First </button>	
								<button class="btn ripple ot-btn-primary" ripple-for="dark"> Second </button>
								</div>
								<!--- Ripple for colored vertical --->
								<div class="btn-grp-vertical">
								<button class="btn ripple cl-btn-primary" ripple-for="light"> First </button>	
								<button class="btn ripple cl-btn-primary" ripple-for="light"> Second </button>
								</div>
								<!--- Ripple for outline vertical --->
								<div class="btn-grp-vertical">
								<button class="btn ripple ot-btn-primary" ripple-for="dark"> First </button>	
								<button class="btn ripple ot-btn-primary" ripple-for="dark"> Second </button>
								</div>

Vertical Button Group

Vertical aligned group are more usefull for menubar, sidebar, linkbar.

First add .btn-grp-vertical class to create the vertical button group and simply add button class (.btn .cl-btn-*, .btn .ot-btn-*, .btn .btn-link) to create the button group.

								<!--- colored vertical example --->
								<div class="btn-grp-vertical">
								<button class="btn cl-btn-primary"> First </button>	
								<button class="btn cl-btn-primary"> Second </button>
								<button class="btn cl-btn-primary"> Third </button>
								</div>
								<!--- outline vertical example --->
								<div class="btn-grp-vertical">
								<button class="btn ot-btn-primary"> First </button>	
								<button class="btn ot-btn-primary"> Second </button>
								<button class="btn ot-btn-primary"> Third </button>
								</div>
								<!--- link vertical example --->
								<div class="btn-grp-vertical">
								<button class="btn btn-link"> First </button>	
								<button class="btn btn-link"> Second </button>
								<button class="btn btn-link"> Third </button>
								</div>

Vertical Group is just another styling for group.
Vertical Group have same capability as other button groups so you can set all the styles include colored, outlined, linked, ripple effects, group sizing easily.

Sizing

Sizing is more important thing to adjust the layout alignments and styles.

You need to add the class with .btn class
Add small class for small size.
Add large class for large size.
Add exlarge class for extra large size. 

								<!--- small size --->
								<div class="btn-grp">
								<button class="btn small cl-btn-primary"> First </button>	
								<button class="btn small cl-btn-primary"> Second </button>
								<button class="btn small cl-btn-primary"> Third </button>
								</div>
								<!--- normal size --->
								<div class="btn-grp">
								<button class="btn cl-btn-primary"> First </button>

All the buttons have some specific states that use in like active, hover, disabled states that we style it with proper mixing so you can use it with just a class.

Hover state

Hoverable states is highlighted the buttons or links easily.

You need to add the .hover class. 

								<!--- All button apply hover --->
								<div class="btn-grp">
								<button class="btn hover cl-btn-primary"> First </button>	
								<button class="btn hover cl-btn-primary"> Second </button>
								<button class="btn hover cl-btn-primary"> Third </button>
								</div>
								<!--- First button apply disabled --->
								<div class="btn-grp">
								<button class="btn hover cl-btn-primary"> First </button>	
								<button class="btn cl-btn-primary"> Second </button>
								<button class="btn cl-btn-primary"> Third </button>
								</div>

Nesting

You can set muliple styles in single button group with buttons.

You can use cl-btn-*, ot-btn-* and ripple class in one button group for nesting.
You can also use states as hover, active and disabled class for highlight the buttons

Outline button group not supported default button and light button style. 

								<!--- Example of single style nesting --->
								<div class="btn-grp">
								<button class="btn cl-btn-dark"> First </button> <!-- dark button -->
								<button class="btn cl-btn-secondary"> Second </button> <!-- default button -->
								<button class="btn cl-btn-light"> Third </button> <!-- light button -->
								</div>
								<!--- Example of multiple nesting --->
								<div class="btn-grp">
								<button class="btn cl-btn-primary"> First </button> <!-- colored button -->
								<button class="btn cl-btn-primary"> Second </button> <!-- colored button -->
								<button class="btn ot-btn-primary"> Third </button> <!-- outlined button -->
								</div>
								<!--- Example of multiple nesting with ripple --->
								<div class="btn-grp">
								<button class="btn cl-btn-primary"> First </button> <!-- colored button -->
								<button class="btn cl-btn-primary"> Second </button> <!-- colored button -->
								<button class="btn ripple ot-btn-primary" ripple-for="dark"> Third </button> <!-- ripple outline button -->
								</div>
								<!--- Example of multiple nesting with state --->
								<div class="btn-grp">
								<button class="btn active cl-btn-primary"> Second </button> <!-- active -->
								<button class="btn hover cl-btn-primary"> First </button> <!-- hoverable -->
								<button class="btn disabled cl-btn-primary"> Third </button> <!-- disabled -->
								</div>
