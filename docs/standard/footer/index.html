Footer

A footer typically contains information about the author of the section, copyright data or links to related documents.

Use the <footer class="footer"> tag to create the footer element.

Example

Footer is a part of navigation bar that holds the links, forms, buttons, icons, compony info and more documentation with responsive layout.

The below footer is an example of butterflycss designed footer which holds tha data of links, buttons, compony information, copyright content, license document and icons.

Footers can hold multiple components, elements, modules with layout. Call to action section is a part of footer which stores the subscribe form, register form, news letter form or buttons.

Copyrights

Copyrights hold the authors name, company rights, license, source data, year of publication and more.

Copyrights © 2021: Butterflycss.

								<footer class="footer bg-light text-center">
								  <div class="Sans text-muted pd1">
								    Copyrights© 2021: 
								    <a class="text-black d-inline-block">Butterflycss.</a>
								  </div>
								</footer>

The footer sets display to the block so we need set the layout of anchor tag, span tag, p tag to inline block for one line. Also you can add icons links in copyrights.

You can change the background color as well as text color with multiple options.

Links

Links are more preferred for footers, navigations, social media links and more.

You need to add .link-data class to apply link effect. It is only working with anchor tag.
