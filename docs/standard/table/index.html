Alertbox

Provide valuable, actionable, and informable feedback or message for the users with flexible range, colors and types for it.

Use the .alertbox class for alertbox.

Colored Alert box

The colored alerts provide colorfull messages and custom colors for alert.

You need to use .colored-alert-* class to apply colored alertbox.

Closing alert Box

Flexbile component for dismissable alerts to popup the message and feedback with closable alert.

You need to use alert-dismiss class to apply closable alertbox. You can use any classes for this alertbox.
It use onclick event for closing the button.
