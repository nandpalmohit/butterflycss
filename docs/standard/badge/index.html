Badge

More informative for labelling and counting component.

Use the .badge class for badges.

Colored Badges

Colored badges are designed with multi color options.

You need to use .cl-badge-* class with color name to apply.

Default
Primary
Secondary
Dark
Light
Warning
Danger
Success
Info

								<div class="badge cl-badge-dark">...</div>
								<div class="badge cl-badge-light">...</div>

Rounded Badges

The radius of badge is dependent on the layout and designs, You have a multiple choices for radius.

You need to use .rounded-* class with multiple choices for rounded badges.

Rounded-10
Rounded-20
Rounded-30

								<div class="badge cl-badge-primary rounded-10">...</div>
								<div class="badge lt-badge-primary rounded-20">...</div>
								<div class="badge cl-badge-primary rounded-30">...</div>

Sizing

The badge size is more important thing for the badge with its fix size.

The badge provide .smallfor small size badges,.largefor large size badges. No need to add the class for regular size badges.

Small
Regular
Large

								<div class="badge small cl-badge-primary">Small</div>
								<div class="badge cl-badge-primary">Regular</div>
								<div class="badge large cl-badge-primary">Large</div>
