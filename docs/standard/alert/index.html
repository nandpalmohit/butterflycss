Alertbox

Provide valuable, actionable, and informable feedback or message for the users with a flexible range, colors and types for it.

Use the .alertbox class for alertbox.

Closing Alert Box

Flexible component for dismissable alerts to popup the message and feedback with closable alert.

You need to use .alert-dismiss class to apply closable alertbox. You can use any classes for this alertbox.
It uses onclick() event for closing the button.

Note -This is colored dismissable alertbox.
Note -This is light dismissable alertbox.

Note -

This is form dismissable alertbox.

								<div class="alertbox form-alert-primary alert-dismiss">
								<h3>Note -</h3>
								<p>This is form dismissable alertbox.</p>
								<button class="close" onclick="this.parentElement.style.display='none';"> ✖ </button>
								</div>
