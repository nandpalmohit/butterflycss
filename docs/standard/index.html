Class

Every component has some primary classes which you need to add first and there are some sizes and colors available for each components.

For example, The button uses .btn class as the main class and simply adds .cl-btn-* class for color options. The primary classes mainly used for creating the design, layout, model and mixing of elements with multiple sizes and color options.

The available color options for the components are default, primary, secondary, warning, danger, success, info, dark, light. All the components use these colors as primary colors.

Some components provide different different sizing options to apply it as per layout-based. The available size options are exsmall, small, normal, large, exlarge.
