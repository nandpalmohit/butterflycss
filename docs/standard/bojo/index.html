Bojo

Lightweight and flexible component to represent the pages, sections, marketing and messages.

Use the .bojo class for Bojo.

Example

It represents the pages as a banner, cover.

The bojo provides .bojo class with 100% of width and you need to use it for creating the bojo component. Also, you can add multiple components and elements with a different range of sizes, styles and colors.

Hello There !

You can create anything anywhere with flexible and make an attractive page.

								<div class="bojo text-center bg-light">
								  <h1 class="Sans-Black size-46 mb1"> Hello There ! </h1>
								  <p class="Sans-Medium size-18 pb1"> You can create anything ... </p>
								  <hr class="type-1">
								  <button class="btn cl-btn-primary Sans mt1"> Learn More </button>
								</div>

Sizing

It covers different different sizing bojo.

The bojo provide .sub-bojofor 80% size bojo,.div-bojofor 90% size bojo. No need to add the class for 100% size bojo.

.sub-bojo covers 80% of width for your page.

.div-bojo covers 90% of width for your page.

.bojo covers 100% of width for your page.

								<!-- sub bojo -->
								<div class="bojo sub-bojo text-center">
								<p class="Sans size-18"> It covers 80% of width for your page. </p>
								</div>
								<!-- div bojo -->
								<div class="bojo div-bojo text-center">
								<p class="Sans size-18"> It covers 90% of width for your page. </p>
								</div>
								<!-- bojo -->
								<div class="bojo text-center">
								<p class="Sans size-18"> It covers 100% of width for your page. </p>
								</div>
