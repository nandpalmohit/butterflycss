Card

Responsive Cards are a more flexible and easy way to use them in multiple options, styles, colors and more.

Use the .card class to create the card.

Card Example

The Card contains multiple classes and methods to create a custom element used as an easy showing of contents that comprises various types of titles, images, buttons, texts.

.card class is neccessary to create a Card component.

The most prolific creation in recent times, the Statue of Unity, is the World’s Tallest Monument. It represents more than a colossal structure facing the Sardar Sarovar Dam on the river Narmada. It is an iconic symbol of the ‘Iron Man’, who played an important role during India’s freedom struggle and thereafter in the unification of the princely states.

Simple Card

The Card contains 100% Width of its parent class. Card contain mulyiple element, styles, text, component.

You need to add .card class to create a card component first.
.card-body class to add texts, links, images in card body.
.card-title class for add title with h1, h2, h3, h4, h5, h6, div class,
.card-subtitle class to add subtitle, badge with h1, h2, h3, h4, h5, h6, p, div class,
.card-text class to add descriptions, text of card with p, div class,
.card-link class to add buttons, links of card with button, link class. You can use <hr> tag for horizontal line.
.br-* class mainly use to set 1px solid border with default, primary, secondary, dark, light, warning, danger, success, info color options.

Image Card

Image card is mainly use to set an image and add some text to describe the image. Mainly use for Profile-card, Product-card, Menu-card, Vertical-card and more card design.

You just need to add <img> tag under the .card class and before the .card-body class.
Add .shadow class to apply box-shadow to card.
We can set card width by using style="width: 450px;" attribute.

Overlay Card

Overlay card mainly use to set image as background for card. The card body set to overlay card for text.

Create a card component and add .overlay-img class in <img> tag to set image for card.
After add attribute in .card-bodyclass for body background.
Add body-overlay = "transparent" attribute to set background for card body.
Add body-overlay = "dark" attribute to set dark background(with 0.3 opacity) for card body.
Add body-overlay = "light" attribute to set light background(with 0.3 opacity) for card body.
This will help when we set any colored image, saturated image or some other images that impact to the text.

Transparent Overlay
add body-overlay="" attribute in card-body class to set body overlay to transparent for body background.
Dark Overlay
add body-overlay="" attribute in card-body class to set body overlay to dark for body background.
Light Overlay
add body-overlay="" attribute in card-body class to set body overlay to light for body background.

								<!--- transparent overlay --->
