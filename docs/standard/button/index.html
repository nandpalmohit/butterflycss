Button

Use it to perform action in pages, forms and more with multiple styles, colors and sizing.

Use the .btn class for every single button.

Button Example

Responsive Buttons indicate a possible user action with predefined styles.

You need to add .btn class to apply Button styles.

Link 

								<button class="btn cl-btn-primary">...</button>
								<a class="btn cl-btn-primary">...</a>
								<input type="button" class="btn cl-btn-primary" value="">
								<input type="submit" class="btn cl-btn-primary" value="">
								<input type="reset" class="btn cl-btn-primary" value="">
You can use all properties and styling with multiple tags include a button, input components that support events and button appearance.

Colored Buttons

Colored Buttons are a more attractive component.

You need to use .cl-btn-* class to apply.

								<button class="btn cl-btn-secondary">...</button>

								<button class="btn cl-btn-danger">...</button>
								<button class="btn cl-btn-success">...</button>
								<button class="btn cl-btn-info">...</button>

Outline Buttons

Link Buttons

After all link is also main purpose for use as reference, links, informative so we build link button.

You need to use .btn-link class to apply.

Link 

								<button class="btn btn-link">button link</button>
								<a class="btn btn-link">link</a>

The link buttons are more usefull to add links, references, social connectivity and more so we create this class for <button> and <a> tag.
In <button> tag set transparent background color as default but set the background color for hover and active state.
In <a> tag set transparent background color as default for all the state.

Ripple Buttons

Ripple Buttons are part of modern design trend. The Button presenting effect.

You need to add .ripple class to apply Ripple effect.
Add ripple-for="dark" attribute for dark ripple effect.
Add ripple-for="light" attribute for light ripple effect. 

								
								<button class="btn ripple btn-default" ripple-for="dark">...</button>
								<button class="btn ripple btn-primary" ripple-for="light">...</button>
								<button class="btn ripple btn-secondary" class="light">...</button>
								<button class="btn ripple btn-dark" ripple-for="light">...</button>
								<button class="btn ripple btn-light" ripple-for="dark">...</button>
								<button class="btn ripple btn-warning" ripple-for="light">...</button>
								<button class="btn ripple btn-danger" ripple-for="light">...</button>
								<button class="btn ripple btn-success" ripple-for="light">...</button>
								<button class="btn ripple btn-info" ripple-for="light">...</button>
								<button class="btn ripple btn-link" ripple-for="dark">...</button>
								
								<button class="btn ripple ot-btn-default" ripple-for="dark">...</button>
								<button class="btn ripple ot-btn-primary" ripple-for="dark">...</button>
								<button class="btn ripple ot-btn-secondary" ripple-for="dark">...</button>
								<button class="btn ripple ot-btn-dark" ripple-for="dark">...</button>
								<button class="btn ripple ot-btn-warning" ripple-for="dark">...</button>
								<button class="btn ripple ot-btn-danger" ripple-for="dark">...</button>
								<button class="btn ripple ot-btn-success" ripple-for="dark">...</button>
								<button class="btn ripple ot-btn-info" ripple-for="dark">...</button>

Ripple effect only allow button component to use it so button makes it as a unique effect with modern design trend. Also note that buttons have different styling and coloring so add ripple with its effect which is dark or light to create more attractive buttons.

Sizing

								<button class="btn cl-btn-primary exlarge block">Extra Large size block</button>
								<button class="btn ot-btn-primary small block">Small size block</button>

The combination of icons and buttons creates multiple styling.

Icon Example

The icons contains multiple styling to choose the right styling with the use of buttons, links, effects, colors and sizing.

Add .iconto create the icon. You can use it with <div>, <span>, <a>, <p> and more tags easily.

In the example below, we use Font Awesome 6 icons. You can use it or download it for free.

								<!-- div tag -->
								<div class="icon">
								<i class="fab fa-google" aria-hidden="true"></i>
								</div>
								<!-- span tag -->
								<span class="icon">
								<i class="fab fa-google" aria-hidden="true"></i>
								</span>
								<!-- a tag -->
								<a class="icon">
								<i class="fab fa-google" aria-hidden="true"></i>
								</a>
								<!-- p tag -->
								<p class="icon">
								<i class="fab fa-google" aria-hidden="true"></i>
								</p>

All the buttons have some specific states use in like active, hover, disabled states and we style it with proper mixing so you can use it with just a class.

Hover State

Button contains hover state which enable when cursor hovers to any links, buttons.

You need to add .hover class in the button tag 

								<!--- Hover state button -->
								<button class="btn cl-btn-default hover">Default<button>

Active State

Button contains active state which enable when any links, buttons are pressed.

You need to add .hover class in the button tag 

								<!--- active state button -->
								<button class="btn cl-btn-default active">Default<button>

Disabled State

Button also contain disabled state which look inactive button.

You need to add .disabled class in the button tag. 

								<!--- disabled state button -->
								<button class="btn cl-btn-default disabled">Default<button>
