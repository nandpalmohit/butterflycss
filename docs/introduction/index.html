Introduction to Butterflycss

The free and open-source library with modern and new generation designed component based css.

Beautifull components that made up with modern design and precompiled css.

Browse Component

Quick Introduction

Butterflycss is purely made with CSS files and doesn't need any javascript files for the project. The library is precompiled with SCSS files.
You just need to care about butterfly.min.css which is a compressed CSS file while butterfly.css is main file where you can customize the docs and set your own goal.

This is the most basic of butterflycss: We created each component and layout-based CSS files so you can easily use it and build a strong and styled component. We provide compiled CSS files as butterfly.css or _.css files as well as we provide compiled and minified CSS files butterfly.min.css. The _.css.map files also available for use with certain browser's developer tools.

If you want to just add minified and compiled files than add butterfly.min.css file to your project or just add compiled file butterfly.css.
