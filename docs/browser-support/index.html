Supported browser

Butterflycss is designed and tested on the latest version of Chrome, Firefox, Edge and Safari. Butterflycss does not support Internet Explorer.

Mobile device

It supports the latest versions of each platform's default browser. Note that proxy browsers (such as Opera Mini, UC Browser and others) are not supported

Laptop device

It supports the latest versions of each platform's default browser. Note that proxy browsers (such as Opera Mini, UC Browser and other) are not supported
