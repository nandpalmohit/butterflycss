Checkbox

Butterfly Forms is designed modern checkbox as a new generation form controls.

A checkbox is one of the HTML forms used on every websites. We're styled a simple checkbox to the next generation checkbox with the help of .checkbox-group class and you can customize the colors, styling, layout and more with other classes.

Overview

Our checkbox is works differently as we use <input> tag and <label> tag for proper styling and you need to specify id and for attribute to relate the <input> tag and <label>. You can use checked and disabled attribute for it.

You need to apply .checkbox-group class for checkbox and apply .checkbox-label to the <label> tag.

.checkbox-group class follows flex property and display the content in flex. If you need to set the content to the center, just add .cn-center class which set the content center. 

								<!--- first checkbox --->
								<div class="checkbox-group">
								  <input type="checkbox" id="defaultCheckbox" name="check">
								  <label for="defaultCheckbox" class="checkbox-label">Default Checkbox</label>
								</div>
								<!--- checked checkbox --->
								<div class="checkbox-group">
								  <input type="checkbox" id="checkedCheckbox" name="check" checked>
								  <label for="checkedCheckbox" class="checkbox-label">Checked Checkbox</label>
								</div>

Inline

.checkbox-group-inline class follows inline-flex property and display the content in inline.

You need to apply .checkbox-group-inline class for checkbox. 

								<!--- first checkbox --->
								<div class="checkbox-group-inline">
								  <input type="checkbox" id="firstCheckbox" name="check">
								  <label for="firstCheckbox" class="checkbox-label"> First Checkbox </label>
								</div>
								<!--- second checkbox --->
								<div class="checkbox-group-inline">
								  <input type="checkbox" id="secondCheckbox" name="check">
								  <label for="secondCheckbox" class="checkbox-label"> Second Checkbox </label>
								</div>
								<!--- checked checkbox --->
								<div class="checkbox-group-inline">
								  <input type="checkbox" id="checkedCheckbox" name="check" checked>
								  <label for="checkedCheckbox" class="checkbox-label"> Checked Checkbox </label>
								</div>

Disabled

Checkbox contain disabled state which look inactive checkbox.

You need to add disabled attribute with .checkbox-group class. 

								<!--- default checkbox --->
								<div class="checkbox-group">
								  <input type="checkbox" id="defaultCheck" name="check">
								  <label for="defaultCheck" class="checkbox-label"> Default Checkbox </label>
								</div>
								<!--- checked checkbox --->
								<div class="checkbox-group">
								  <input type="checkbox" id="checkedCheck" name="check" checked>
								  <label for="checkedCheck" class="checkbox-label"> Checked Checkbox </label>
								</div>
								<!--- disabled checkbox --->
								<div class="checkbox-group" disabled>
								  <input type="checkbox" id="disabledCheck" name="check">
								  <label for="disabledCheck" class="checkbox-label"> Disabled Checkbox </label>
								</div>
								<!--- disabled checked checkbox --->
								<div class="checkbox-group" disabled>
								  <input type="checkbox" id="disabledCheckedCheck" name="check" checked>
								  <label for="disabledCheckedCheck" class="checkbox-label"> Disabled Checked Checkbox </label>
								</div>
