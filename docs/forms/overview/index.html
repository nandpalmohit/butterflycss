Forms

Butterfly Forms is predesigned and fully styled as new generation form components. Guidelines for form styles, layout options, custom components and wide variety of examples.
The basic use of form is to collect data from user and interact with user. Be sure to have basic knoweldge of input types.

HTML5 supports new type of form controls, such as date and time, email, number, search etc.
Butterfly supports the following native HTML form elements: <form>, <input>, <button>, <label> and <textarea>.

  • Checkbox
  • File
  • Input fields
  • Select
  • Radio
  • Range

Basic Example

Here's quick example to demonstrate the styles and layout. The form controls and layout creates responsive form design.

The below example is simple login form which include email and password input fields with one button and one link.
We'll never share your email with anyone else.
Forgot Password? 

								<form>
								  <!--- email input --->
								  <div class="form-control">
								    <label for="ExampleEmail"> Enter Email </label>
								    <input type="email" class="form-input" id="ExampleEmail" name="email">
								    <small class="text-secondary" id="emailHelp">We'll never share your email with anyone else.</small>
								  </div>
								  <!--- password input --->
								  <div class="form-control">
								    <label for="ExamplePass"> Enter Password </label>
								    <input type="Password" class="form-input" id="ExamplePass" name="password">
								  </div>
								  <!--- forget link --->
								  <div class="form-control my2 text-center">
								    <a class="btn btn-link"> Forgot Password? </a>
								  </div>
 								  <!--- login button --->
								  <div class="form-control text-center">
								    <button class="btn cl-btn-primary ripple"> Login to your account </button>
								  </div>
								</form>

Signup Example

It display the signup form with the use of layout and addition register buttons.

The below example is simple signup form which include text, email, tel, number input fields with multiple links and buttons.
We'll never share your email with anyone else.
Enter atleast 8 character for your password.
Already a Member? Login Now 

								<form>
								  <div class="row">
								    <div class="col-5">
								      <!--- text input --->
								      <div class="form-control">
								        <label for="ExampleFname"> First Name </label>
								        <input type="text" class="form-input" id="ExampleFname" name="fname">
								      </div>
								    </div>
								    <div class="col-5">
								      <!--- text input --->
								      <div class="form-control">
								        <label for="ExampleLname"> Last Name </label>
								        <input type="text" class="form-input" id="ExampleLname" name="lname">
								      </div>
								    </div>
  								  </div>
								  <!--- email input --->
								  <div class="form-control">
								    <label for="ExampleEmail"> Enter Email </label>
								    <input type="email" class="form-input" id="ExampleEmail" name="email">
								    <small class="text-secondary" id="emailHelp">We'll never share your email with anyone else.</small>
								  </div>
								  <!--- password input --->
								  <div class="form-control">
								    <label for="ExamplePass"> Enter Password </label>
								    <input type="Password" class="form-input" id="ExamplePass" name="password">
								    <small class="text-secondary" id="passHelp">Enter atleast 8 character for your password.</small>
								  </div>
								  <!--- condition checkbox --->
								  <div class="form-control my3 text-center">
								    <div class="checkbox-group">
								      <input type="checkbox" id="ExampleCond">
								      <label class="checkbox-label" for="ExampleCond"> Allow all the terms and conditions.</div>
								    </div>
								  </div>
 								  <!--- signup button --->
								  <div class="form-control text-center">
								    <button class="btn cl-btn-primary block ripple"> Sign up for free </button>
								  </div>
 								  <!--- login button --->
								  <div class="form-control text-center">
								    <a class="btn btn-link"><span class="text-dark"> Already a Member? </span> Login Now. </a>
								  </div>
								</form>
