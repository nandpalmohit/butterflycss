Use the .form-* class for creating a structure for it.

Sign In

The below example is simple sign in form which include email and password input fields with one button and one link.

Use the .form-control class for simple input field styling. It uses label for input fields with a small class that pass a small message for inputs.

We'll never share your email with anyone else.
Forgot Password? 

								<form>
								  <!--- email input --->
								  <div class="form-control">
								    <label for="SignInEmail"> Enter Email </label>
								    <input type="email" class="form-input" id="SignInEmail" name="email">
								    <small class="text-secondary" id="emailHelp">We'll never share your email with anyone else.</small>
								  </div>
								  <!--- password input --->
								  <div class="form-control">
								    <label for="SignInKey"> Enter Password </label>
								    <input type="Password" class="form-input" id="SignInKey" name="password">
								  </div>
								  <!--- forget link --->
								  <div class="form-control my2 text-center">
								    <a class="btn btn-link"> Forgot Password? </a>
								  </div>
 								  <!--- login button --->
								  <div class="form-control text-center">
								    <button class="btn cl-btn-primary ripple"> Login to your account </button>
								  </div>
								</form>
