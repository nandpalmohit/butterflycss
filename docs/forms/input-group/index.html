Input Group

Butterfly Froms created layout based input groups by adding texts, buttons, links with inputs, files, selects.

You need to apply .form-group class.

Example

Input Group is custome builed form component that is allow to use <label>, <input>, <button> and texts.

You need to apply .form-group class first. Add .group-label class with the <label> tag and .group-input class with the <input> tag

Input Groups support all the input types, textarea, select, checks and layout with modern customization.

								<!--- required input --->
								<div class="float-group">
								  <input type="text" class="float-input" id="floatName" name="fullname" required>
								  <label for="floatName" class="float-label">Your Full Name</label>
								</div>
								<!--- not-required input --->
								<div class="float-group">
								  <input type="text" class="float-input" id="floatName2" name="fullname">
								  <label for="floatName2" class="float-label">Your Full Name</label>
								</div>

Sizing

Floating inputs have small to large size set.

Add .small class with .float-group class for small sizing.
Add .large class with .float-group class for large sizing. 

								<!--- small input --->
								<div class="float-group small">
								  <input type="text" class="float-input" id="smallSize" name="fullname" required>
								  <label for="smallSize" class="float-label">Upload Your File</label>
								</div>
								<!--- normal input --->
								<div class="float-group">
								  <input type="text" class="float-input" id="normalSize" name="fullname" required>

Disabled

Floating Input contain disabled state to fill light colors and opacity for inputs and made a disabled type input.

You need to apply disabled attribute with .float-group class.
You don't need to add disabled in <input>'s. 

								<!--- empty input --->
								<div class="float-group" disabled>
								  <input type="text" class="float-input" id="disabled1" name="fullname" required>
								  <label for="disabled1" class="float-label">Disabled Input Label Text</label>
								</div>
								<!--- filled required --->
								<div class="float-group" disabled>
								  <input type="text" class="float-input" id="disabled2" name="disabled2" value="you can't access this inputs.">
								  <label for="disabled2" class="float-label">label</label>
								</div>

ReadOnly

Floating Input contain readonly attribute to show an input-field is read-only.

You need to apply readonly attribute with <input> tag. 

								<!--- empty input --->
								<div class="float-group">
								  <input type="text" class="float-input" id="floatName" name="fullname" readonly>
								  <label for="floatName" class="float-label">Your Full Name</label>
								</div>
								<!--- filled input --->
								<div class="float-group">
								  <input type="text" class="float-input" id="floatName2" name="fullname"  value="Mohit Nandpal" readonly>
								  <label for="floatName2" class="float-label">Your Full Name</label>
								</div>

Wrapping Input

Butterfly Forms are also designed latest styling of Wrap Inputs fields. Wrap input fields are covered with multiple colors, sizes, layouts and more.

You need to apply .wrap-group class.

Example

Wrapping Input is combination of <input> tag and <label> tag with its own classes and it is designed with the newer inputs. It is mainly designed as a wrapping of input fields with some text.

You need to apply .wrap-group class first. Add .wrap-input to the <input> tag and apply .wrap-label to the <label> tag. 

								<!--- required input --->
								<div class="wrap-group">
								  <input type="text" class="wrap-input" id="wrapName" name="fullname" required>
								  <label for="wrapName" class="wrap-label">Your Full Name</label>
								</div>
								<!--- not-required input --->
								<div class="wrap-group">
								  <input type="text" class="wrap-input" id="wrapName2" name="fullname">
								  <label for="wrapName2" class="wrap-label">Your Full Name</label>
								</div>
