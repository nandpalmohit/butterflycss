File

Butterfly Forms is designed modern file as a new generation form controls.

You need to apply .file-group class.

Example

File have a different class as you need to specify .file-group class to create a layout and design for it. The perfect combination of <input> ~ <label> tag with its own classes.

You need to apply .file-group class for file. Add .file-input to the <input> tag and apply .file-label to the <label> tag.

.file-group class follows grid property and display the content in column. 

								<!--- file input --->
								<div class="file-group">
								  <input type="file" class="file-input" id="docFile" name="doc">
								  <label for="docFile" class="file-label">Upload Your File</label>
								</div>

Multiple Files

Files allowed multiple documents to submit at a one time.

You need to apply multiple attribute in <input> tag. 

								<!--- file input --->
								<div class="file-group">
								  <input type="file" class="file-input" id="multiDocFile" name="doc" multiple>
								  <label for="multiDocFile" class="file-label">Upload Your File</label>
								</div>

Sizing

File sizes is important to set layout and design as per form sizes.

You need to add .small class for small sizing, .large class for large sizing. 

								<!--- small file input --->
								<div class="file-group small">
								  <input type="file" class="file-input" id="docFile" name="doc">
								  <label for="docFile" class="file-label">Upload Your File</label>
								</div>
								<!--- file input --->
								<div class="file-group">
								  <input type="file" class="file-input" id="docFile" name="doc">
								  <label for="docFile" class="file-label">Upload Your File</label>
								</div>
								<!--- large file input --->
								<div class="file-group large">
								  <input type="file" class="file-input" id="docFile" name="doc">
								  <label for="docFile" class="file-label">Upload Your File</label>
								</div>

Disabled

Files are more usefull with the states as disabled.

You need to apply disabled attribute with associated .form-group class. 

								<!--- file input --->
								<div class="file-group" disabled>
								  <input type="file" class="file-input" id="docFile" name="doc">
								  <label for="docFile" class="file-label">Upload Your File</label>
								</div>
