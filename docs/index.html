Quick Start

Looking to quickly add Butterflycss to your project? We have multiple options by using .css file.
The css file named as butterfly.css, which include all the components, elements, forms, layouts, modules within one file. You can access this all scss files by github pages.

The butterflycss is purely precompiled code and we created less css file which is butterfly.min.css. If you want to customize the code than you need to include butterfly.css.

We build the library with pure css files so we don't need to import any javascript file in the project.

You need only 1 file to handle css to use Butterflycss.

Butterflycss is a CSS Library.This means it provides CSS classes to help you style your HTML code.
To use Butterflycss, you can use the pre-compiled .css file either link with your HTML page or download the Source code.

CSS

Import the css file directly from jsDelivr.

							
								@import"https://unpkg.com/tailwindcss@^2/dist/tailwind.min.css";

HTML Link

Import the css file directly from jsDelivr.

							
								<link href="https://unpkg.com/tailwindcss@^2/dist/tailwind.min.css">

HTML starter template

To use the library, Be sure to have your pages set up with the latest design and development standard. It means you have to work with the HTML5 doctype and include the responsive viewport meta tag to properly handle responsive styles on all the device.

							
								<!doctype HTML>
								<html>
								<head>
								  <!--- required meta tags for responsive --->
								  <meta charset="UTF-8">
								  <meta name="viewport" content="width=device-width, initial-scale=1.0">
								  <!--- Butterflycss link --->
								  <link href="https://unpkg.com/tailwindcss@^2/dist/tailwind.min.css" rel="stylesheet">
								  <!---...--->
								</head>
								<body>
								  <h1>Hello World!</h1>
								</body>
								</html>
