Build Free, Responsive sites with Butterflycss

It's lightweight and free open source library to create a responsive design for customize mobile-sites and tablet-sites with Butterfly. It creates with scss variables, responsive layout design, components, elements, modules and more.

Layout

Build Responsive Layout for Mobile-First webpages.

Create your website with responsive layout design and Mobile-First webpages with the use of layouts, components, elements, modules and custom classes.

Class

An Utility Classes for custom use.

Utility classes help you to work faster and instance for the webpages. It contains colors, margins, paddings, spacings, shadows, hr's, typography and everything that you can make up the well design.

Components

Flexible Components that change everything.

The Components make the webpages creative and responsive by using combintations of the components and colors. We provides alerts, badges, bojo, buttons, groups, cards, footers and more to create a full websites.

Modules

Forms for Multi Purpose Collections

Its enough to get new and an attractive designs for the input forms as well as input sections for data. The new design of input types, checkboxes, radio buttons, files, ranges and switches thay feel you powerfull for it.

Navbar

The New Navbars

It just designed as per maximum use of layout that creates responsive layout for the Navbars. Multiple navbars that look are so positionable with relative, fixed or sticky with spacing of good.

Table

The Data Entry Tables

Data is storing in the table so it is also be a main part of the layout that colors, designs matters for it. Multiple tables that look with the Headings, Body, Rows, Colums and Caption for it.
